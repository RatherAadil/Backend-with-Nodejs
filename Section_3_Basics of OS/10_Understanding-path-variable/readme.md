# Absolute Path:

- In windows it starts from C drive.
- Its the path from root to that specific file.
  Ex:

```
C:\Users\ADIL\Desktop\Backend with Node Js\Section_3_Basics of OS\Understanding-path-variable\hello.txt
```

# Relative Path:

Starts from that specific file or folder.
Ex:

```
Section_3_Basics of OS\Understanding-path-variable\hello.txt
```

## utility function:

To convert path into windows path

```
cygpath -w 'C:\Users\ADIL\Desktop\Backend with Node Js\Section_3_Basics of OS\Understanding-path-variable\hello.txt'
```
