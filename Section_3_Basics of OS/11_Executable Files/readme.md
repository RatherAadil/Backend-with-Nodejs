# Types

- Script Executable files
- Binary Executable files

## Script Executable files

- Any file that contains a command is known as script executable file

* To execute the commands in text.txt file we will use

```
./text.txt
```

- abc.sh is a linux executable file

## Binary Executable files

- Dont contain any normal text

* extension .exe
